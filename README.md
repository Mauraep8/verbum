# VERBUM



## About the Project

Verbum is a language app that generates conjugation exercices for students learning French. Users can customize their exercices to their level and needs.

## Built With
- HTML
- SASS
- JavaScript
- React
- Redux.js
- Node.js
- Prisma
- MySQL
- Docker
